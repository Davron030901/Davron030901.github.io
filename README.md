# Davron Aliqulov — AI Engineer Portfolio

Personal portfolio website hosted on GitHub Pages.

🌐 **Live site:** https://Davron030901.github.io

## Tech Stack
- Pure HTML5, CSS3, JavaScript (no frameworks)
- JetBrains Mono + Inter fonts (Google Fonts)
- Animated neural network canvas
- Scroll-reveal animations
- Fully responsive (mobile-first)

## Deploy to GitHub Pages

1. Create a new repository named exactly: `Davron030901.github.io`
2. Upload these 4 files to the repo root:
   - `index.html`
   - `style.css`
   - `main.js`
   - `README.md`
3. Go to **Settings → Pages → Source: main branch → / (root)**
4. Site will be live at `https://Davron030901.github.io` in ~2 minutes
